## 📌 About the Tool

The **Sketchbook Pro Activated Tool** provides a fully unlocked and portable version of Autodesk Sketchbook Pro for Windows.  
It’s designed for digital artists, illustrators, and sketchers who need a clean, fast drawing environment without installation or licensing interruptions.

The build is portable, self-contained, and includes all the core features — brushes, layers, perspective tools, and tablet support.

Perfect for use from USB drives, external storage, or shared systems — just launch and start creating.

---

## 🚀 Features

- ✏️ Full brush engine unlocked  
- 🖌️ Pen & tablet input supported  
- 📐 Perspective rulers & symmetry tools  
- 🧰 No installation or activation needed  
- 🧠 Portable and fast to launch  
- ✅ Compatible with Windows 10 & 11  

---

## 🧩 How to Use

1. Extract the downloaded archive  
2. Run `AppLauncher.exe`  
3. Sketchbook opens instantly — ready to draw  
4. Save work anywhere — everything is local
